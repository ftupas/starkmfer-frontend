# eth-amsterdam-front

This is the front-end for [https://github.com/rootulp/eth-amsterdam](https://github.com/rootulp/eth-amsterdam)

## Local Development

1. Clone this repo

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
